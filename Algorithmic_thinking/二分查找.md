# 二分查找

```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        # 二分查找:左闭右开
        left, right = 0, len(nums)
        while left < right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            else:
                right = mid
        return left
```

#### 1.循环条件的不同

- **写法**：`while left < right`
- **为什么不用 `<=` ？**
  - 因为 `right` 是个“虚”的边界。当 `left == right` 的时候，说明区间里已经没有东西了（这就好比 `[2, 2)`，起点是 2，终点也是 2 但不包含 2，里面是空的）。
  - 所以不需要判断 `left == right` 的情况，直接退出。

#### 2. 缩小右边界的不同

- **写法**：`right = mid`
- **为什么不减 1 ？**
  - 因为你的 `right` 本身的定义就是“开区间（不包含）”。
  - 当你发现 `nums[mid] > target`，说明 `mid` 不是我们要找的。
  - 既然 `mid` 不是，那我们就把右边界墙壁挪到 `mid` 这里来。因为右边界本来就不包含，所以设置成 `right = mid` 就等于把 `mid` 排除在外了。





```python
def binarySearch(kind, target):
    """
    在一种零件里找可靠性刚好等于 target 的零件。
    如果找不到，就返回它应该待的位置（变负数），方便后面找“大于 target 的最小值”。
    """
    low = 0
    high = len(kind) - 1
    while low <= high:
        mid = (low + high) // 2  
        if kind[mid].reliability > target:
            high = mid - 1
        elif kind[mid].reliability < target:
            low = mid + 1
        else:
            return mid  # 运气好，刚好找到了可靠性一样的
    return -low - 1  # 没找到，返回 -(插入位置)-1
```



简单来说，当二分查找**没有找到**完全等于 `target` 的值时，它不想只告诉你“没找到”，它还想顺便告诉你：**“虽然我没找到你要的那个数，但如果你非要把它插进这个有序序列，它应该排在哪个位置。”**

------

### 1. 拆解这个公式

- **`low` 的含义：** 在二分查找结束时，如果没找到目标，变量 `low` 停下的位置正好就是**第一个大于目标值**的元素的索引。

- **为什么要 `-1`？** 因为索引可以是 `0`。如果目标应该插在开头（位置 `0`），返回 `-0` 会让程序误以为在索引 `0` 找到了目标（因为 `0` 和 `-0` 是一样的）。

  为了区分“找到了”和“没找到”，我们统一把没找到的情况变成**负数**。

------

### 2. 举个直观的例子

假设我们有一组零件的可靠性：`[50, 70, 90]`（索引分别是 0, 1, 2）。

我们要找可靠性为 **`60`** 的零件：

1. **开始二分：** `low=0`, `high=2`, `mid=1` (值是 70)。
2. **比较：** 70 比 60 大，说明 60 如果存在，应该在左边。`high = mid - 1` (变成 0)。
3. **再比：** `low=0`, `high=0`, `mid=0` (值是 50)。
4. **再比：** 50 比 60 小，`low = mid + 1` (变成 1)。
5. **结束：** 此时 `low(1) > high(0)`，循环停止。

**结果：**

- 没找到 60。
- 函数返回 `-low - 1` $\rightarrow$ `-1 - 1 = -2`。
