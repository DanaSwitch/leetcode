# 算法总结

### 重复出现次数

```python
# <完美走位>
from collections import Counter
import collections


s = input().strip()

n = len(s)
# 目标数量：每个字符应该出现的次数
target = n // 4
# 统计当前字符串中各字符的数量
# Counter 会生成类似 {'A': 2, 'S': 1, ...} 的字典
count = Counter(s)
# 检查是否已经是完美走位
if all(count[c] == target for c in "WASD"):
    print(0)
    return
```
