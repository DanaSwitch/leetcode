# 前缀和模板

### 1. 前缀和的核心思想

前缀和的本质是**空间换时间**。通过预处理一个数组 `pre_sum`，使得任意一段连续区间的和可以快速通过两个端点的值作差得出：

$$Sum(i, j) = pre\_sum[j+1] - pre\_sum[i]$$



### 2. 前缀和模板套路

构建和使用前缀和通常遵循以下四个步骤：

- **定义数组：** 长度通常为 `n+1`，方便处理边界（如从数组第一个元素开始的片段）。

- **构建数组：**

  ```python
  pre_sum = [0] * (n + 1)
  for i in range(n):
  pre_sum[i + 1] = pre_sum[i] + nums[i]
  ```

- **区间查询：** 在任何需要子数组和的地方，直接代入 `pre_sum[j+1] - pre_sum[i]`。

- **优化算法：**  **暴力枚举（你的现状）：** 双重循环，时间复杂度 $O(n^2)$。

  - **哈希表优化（进阶）：** 如果题目要求“和为 K”或者“和满足特定条件”，通常可以用 `Hash Map` 存储前缀和出现的次数或位置，将时间复杂度降至 $O(n)$。
