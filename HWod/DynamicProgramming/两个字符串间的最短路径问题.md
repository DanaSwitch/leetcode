

|                       | **第 j−1 列**              | **第 j 列 (当前列)**     |
| --------------------- | -------------------------- | ------------------------ |
| **`preRow` (上一行)** | `preRow[j-1]` (左上方)     | **`preRow[j]` (正上方)** |
| **`curRow` (当前行)** | **`curRow[j-1]` (正左方)** | `curRow[j]` (正在算的点) |



### `curRow[j]` 的真实含义

在第 `i` 轮循环中，`curRow[j]` 表示：

**从起点 $(0,0)$ 出发，到达坐标点 $(i, j)$ 的最短路径长度。**



函数solve()返回结果，main()调用solve()的时候要打印出来

即：print(solve())才能显示结果
