# DP和DFS的区别

| **特性**     | **DFS (搜索)**                   | **DP (记忆化搜索)**         |
| ------------ | -------------------------------- | --------------------------- |
| **数据流向** | **容器共享/全局变量** (一路向底) | **函数返回值** (层层归并)   |
| **状态记录** | 通常不存，仅关注当前路径         | 必须存入 `memo`，避免重复   |
| **核心逻辑** | 递归 + 回溯 (Backtracking)       | 递归 + 记忆化 (Memoization) |
| **目标场景** | 列举路径、是否存在、逻辑校验     | 方案总数、最优值 (Max/Min)  |
| **思维模型** | 探索者：深入森林，带回记录       | 计算者：拆解问题，缓存答案  |

dfs是从上往下: 用共享变量; dp是从下往上: 所以要返回
