# LongestLowFailurePeriod

```python
import sys # 1. 导入工具箱

# ...
line2 = sys.stdin.readline().strip()   # 2. 读一行字，并去掉首尾看不见的符号
nums = list(map(int, line2.split()))   # 3. 把这行字切开，变成数字，装进列表
```

### 第一步：为什么要 `import sys`？

- **生活比喻**：Python 自带很多功能，但有些“重型工具”放在仓库里。`sys` 就是一个跟 **系统（System）** 打交道的工具箱。
- **作用**：我们需要用这个工具箱里的 `stdin`（Standard Input，标准输入）来读取数据。
- **为什么不用 `input()`？**：虽然平时写代码用 `input()` 比较多，但在做算法题（特别是数据量大时），`sys.stdin.readline()` 比 `input()` **快得多**，就像“批发进货”和“零售”的区别。为了防止超时，很多题解习惯用这个。

------

### 第二步：`sys.stdin.readline().strip()` 是在干嘛？

假设输入的数据是：`0 0 100 2`，然后你按了一下回车键。

1. **`sys.stdin.readline()`**：
   - 它的意思是：“从键盘（或者测试文件）里读取**一整行**文字”。
   - **坑点**：它会把你最后按的那个“**回车键**”也读进去！
   - **实际读到的内容**：`"0 0 100 2\n"` （注意那个 `\n`，它代表换行符）。
2. **`.strip()`**：
   - 它的意思是：“**剥去**”。它会把字符串**开头和结尾**的空白符号（空格、换行符 `\n`、制表符 `\t`）全部删掉。
   - **处理后的内容**：`"0 0 100 2"`。
   - **为什么要做这一步？** 如果不删掉 `\n`，后面处理数字时可能会出乱子，清理干净是好习惯。

------

### 第三步：`list(map(int, line2.split()))` 这句最难懂，咱们拆成三块

这就好比一条 **流水线**，把一长串字符串加工成我们要的数字列表。

#### 1. `.split()` —— 切割工

- **作用**：拿着刀，遇到 **空格** 就切一刀。
- **输入**：`"0 0 100 2"` (这是一个长字符串)
- **输出**：`['0', '0', '100', '2']`
- **注意**：这时候它们还是 **字符串（引号包着的）**，不是数字！Python 认为 `'100'` 和 `'hello'` 是一类东西，不能用来做加减法。

#### 2. `map(int, ...)` —— 转化工

- **作用**：`map` 是“映射”的意思。它的任务是把 `int`（变成整数）这个命令，**应用到后面列表里的每一个元素上**。
- **过程**：
  - 把 `'0'` 变成 `0`
  - 把 `'0'` 变成 `0`
  - 把 `'100'` 变成 `100`
  - ...
- **结果**：它生成了一个“准备好变数字的迭代器”（你看不到具体的列表，它是一个中间状态）。

#### 3. `list(...)` —— 打包员

- **作用**：把上面那个“中间状态”实实在在地转换成一个 **Python 列表**。
- **最终结果**：`[0, 0, 100, 2]`
- **现在**：这才是我们可以进行数学计算的数字列表了。

------

### 总结图解

为了让你更直观，看这个“数据变形记”：

| **步骤**     | **代码**        | **数据长什么样**         | **类型**                |
| ------------ | --------------- | ------------------------ | ----------------------- |
| **原始输入** | (键盘输入)      | `0 0 100 2\n`            | 字符流                  |
| **读取**     | `readline()`    | `"0 0 100 2\n"`          | 字符串 (String)         |
| **清理**     | `.strip()`      | `"0 0 100 2"`            | 字符串 (String)         |
| **切割**     | `.split()`      | `['0', '0', '100', '2']` | 字符串列表              |
| **转化**     | `map(int, ...)` | (等待转化的中间态)       | Map对象                 |
| **打包**     | `list(...)`     | `[0, 0, 100, 2]`         | **整数列表 (Int List)** |
