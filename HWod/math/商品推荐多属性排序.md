# 商品推荐多属性排序

### 思路：

> 为什么使用元组？

在 Python 中，比较两个元组（例如 `(a, b)` 和 `(c, d)`）时，它遵循 **“字典序”**：

- 先比较第一个元素（索引 0）。
- 如果第一个元素相等，再比较第二个元素。
- 这完美契合了题目中“属性 1 相同按属性 2 排”的要求。

这里的 `key=sort_key` 是 Python 中**高阶函数**的一种用法。它的意思是：**“在排序过程中，先不要直接比较两个商品，而是先给每个商品调用一下 `sort_key` 函数，拿函数返回的那个‘结果’去排。”**
