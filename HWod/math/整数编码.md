# 整数编码

## 题目意思

就是把一个数字先转化为二进制，7 bit进行拆分，没拆完就左边拼接1，拆完就拼0，这1B的数据从左往右放（小端存储）



## 代码

```python
def encode_int(n):
    n = int(n)
    result = []
    
    while True:
        # Take the lowest 7 bits
        byte = n & 0x7F
        n >>= 7
        if n > 0:
            byte |= 0x80  # Set high bit: more bytes follow
        result.append(byte)
        if n == 0:
            break
    
    # Output as uppercase hex, each byte zero-padded to 2 digits
    return ''.join(f'{b:02X}' for b in result)

n = input().strip()
print(encode_int(n))
```



> f'{b:02X}'

### 1. `X` — 转换为十六进制（大写）

- **作用**：将数字（十进制）转换为 **十六进制**。
- **大小写**：使用大写的 `X` 会输出 `A, B, C, D, E, F`；如果使用小写的 `x`，则输出 `a, b, c, d, e, f`。
- **示例**：数字 `10` 使用 `X` 会变成 `"A"`。

### 2. `2` — 设定宽度为 2

- **作用**：规定输出的字符串**至少**占用 2 个字符的宽度。
- **示例**：数字 `172`（十六进制为 `AC`）本身就是 2 位，所以不受影响。

### 3. `0` — 左侧补零

- **作用**：如果数字转换后的位数**不足 2 位**，就在左侧用 `0` 填充，直到补满 2 位。
- **为什么重要**：在处理字节流（Byte Stream）时，我们通常希望每个字节都固定显示为 2 位。
