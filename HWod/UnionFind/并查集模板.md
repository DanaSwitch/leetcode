## 并查集

### 简单模板

```python
# 并查集（26个字母）
parent = list(range(26))  # 每个字母初始时老大是自己

# 找老大
def find(x):
    while parent[x] != x:  # 自己是自己的老大, 说明找到根了
        # 先压缩, 再移动
        parent[x] = parent[parent[x]]
        x = parent[x]
    return x

# 字母小的当老大
def union(x, y):
    px, py = find(x), find(y)
    if px != py:
        if px < py:  # 让字典序小的作为根节点
            parent[py] = px
        else:
            parent[px] = py

```
