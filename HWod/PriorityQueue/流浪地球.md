# 流浪地球

## 思路

在这里使用优先队列是因为我们引入了时间这个概念，我们按照时间先后顺序依次点火。

用一个优先队列pq[]来存储所有发动机的启动事件，将(启动时间, 发动机编号)按照启动时间的优先级压入pq[]，最后找到最大的启动时间，将存储在队列pq[]里对应的发动机编号打印出来

### 难点

难点在于关联启动

```python
for neighbor in [(cur_p-1)%N, (cur_p+1)%N]:
    if cur_t + 1 < activate_time[neighbor]:
        activate_time[neighbor] = cur_t + 1
        heapq.heappush(pq, (activate_time[neighbor], neighbor))
```







### 代码重点

过时校验，剪枝

```python
while pq:
    cur_t, cur_p = heapq.heappop(pq)  # 从时刻早的开始
    if cur_t > activate_time[cur_p]:
        continue
```





当你写 `for neighbor in [A, B]:` 时，Python 会：

1. 先计算出 `A` 的值，赋值给 `neighbor`，执行一次循环体。
2. 再计算出 `B` 的值，赋值给 `neighbor`，执行第二次循环体。
