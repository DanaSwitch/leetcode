# 优先队列

## 1. 什么是优先队列？

在计算机科学中，优先队列是一种抽象数据类型。它的核心逻辑是：

- 每个元素都有一个“优先级”。
- **出队**时，总是移除优先级最高（或最低）的元素。
- **入队**时，元素会自动根据优先级找到自己的位置。

### 核心实现：堆（Heap）

虽然你可以用排序列表来实现优先队列，但最高效的方式是使用**二叉堆（Binary Heap）**。

- **小顶堆（Min-Heap）：** 最小的元素总是在根节点（堆顶）。
- **大顶堆（Max-Heap）：** 最大的元素总是在根节点。

------

## 2. Python 中的 `heapq` 模块

Python 的 `heapq` 模块提供了一个**小顶堆**的实现。这意味着：默认情况下，**数值越小，优先级越高**。

### 常用操作速查表

| **函数**                     | **功能**                       |
| ---------------------------- | ------------------------------ |
| `heapq.heappush(heap, item)` | 将元素压入堆中，同时保持堆特性 |
| `heapq.heappop(heap)`        | 弹出并返回堆中**最小**的元素   |
| `heapq.heapify(list)`        | 将一个现有的列表原地转换为堆   |
| `heapq.nlargest(n, iter)`    | 返回迭代器中最大的 n 个元素    |

------

## 3. 代码实战演练

让我们看一个具体的例子：

```python
import heapq

# 1. 创建一个空堆
pq = []

# 2. 添加元素 (heappush)
heapq.heappush(pq, 10)
heapq.heappush(pq, 1)
heapq.heappush(pq, 5)

print(f"当前堆状态: {pq}") # 注意：内部是以数组存储的，并不完全有序，但 pq[0] 永远是最小的

# 3. 弹出最小值 (heappop)
smallest = heapq.heappop(pq)
print(f"弹出的最小值: {smallest}")
print(f"弹出后的堆: {pq}")

# 4. 将现有列表转化为堆 (heapify)
nums = [20, 3, 15, 7]
heapq.heapify(nums)
print(f"Heapify 后的列表: {nums}")
```

------

## 4. 进阶：如何实现“大顶堆”？

因为 `heapq` 默认只支持小顶堆，如果你想让数字大的先出队，有一个非常机智的“黑客”做法：**将所有数值取负数**。

- 存入时：`heapq.heappush(pq, -val)`
- 取出时：`val = -heapq.heappop(pq)`

> **冷知识：** `heapq` 在内部其实就是一个普通的 Python 列表，它只是通过算法确保了列表的第一个元素（`pq[0]`）永远是最小的。
