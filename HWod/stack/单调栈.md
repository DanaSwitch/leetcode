# 单调栈

可以在O(n)的时间复杂度内，找到第一个

每个元素出栈时，遇到让它出栈的新元素，就是它的“下一个更大/更小”的元素



在元素的左边，找到小于它且最接近它的数字

```python
def find_left(lines):
    stack = []
    n = len(lines)
    left = [-1] * n
    for j in range(n):
        # 淘汰掉大于等于我的
        while stack and lines[stack[-1]] >= lines[j]:
            stack.pop()
        if stack:
            left[j] = stack[-1]
        stack.append(j)
    return left
```
