# 日志解析



```python
def check(T, n, defaultCnt, extraCnt, nums):
    # 记录所有系统在 T 秒内，基础处理能力之外，还缺多少工作量
    total_extra_needed = 0
    
    for a in nums:
        # 每个系统在 T 秒内，基础能处理的量是 T * defaultCnt
        # 如果日志量 a 大于这个值，差额就需要额外资源来补
        if a > T * defaultCnt:
            gap = a - T * defaultCnt
            needed = ceil(gap / extraCnt)
            total_extra_needed += needed
    
    # 关键点：所有系统需要的额外资源总秒数，不能超过我们拥有的总时间 T
    return total_extra_needed <= T
```



total_extra_needed += needed

+= 是将所有的额外资源都加在一起，因为这个是共享资源
