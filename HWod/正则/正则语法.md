# 正则语法

### `re.findall(r'\d+', string)` 详解

find all

```python
input1 = input().strip()  # 区间
intervals_raw = list(map(int, re.findall(r'\d+', input1)))
```



```python
re.findall(r'\d+', "[1,10],[15,20],[18,30]")
# 返回: ['1', '10', '15', '20', '18', '30']
```

| 部分         | 含义                                              |
| ------------ | ------------------------------------------------- |
| `re.findall` | 找出字符串中**所有**匹配项，返回列表              |
| `r'...'`     | raw string，反斜杠不转义（`\d` 不会被误读成别的） |
| `\d`         | 匹配任意**数字字符**（0-9）                       |
| `+`          | 前面的元素出现**1次或多次**（贪心，尽量多匹配）   |



### 找数字和字母

| 模式          | 含义                    |
| ------------- | ----------------------- |
| `\d`          | 仅数字 `0-9`            |
| `[a-zA-Z]`    | 仅字母                  |
| `\w`          | 字母+数字+下划线 `_`    |
| `[a-zA-Z0-9]` | 字母+数字（不含下划线） |
