## 正则表达式



| **^ 的位置**          | **角色**   | **含义**             | **例子**                                 |
| --------------------- | ---------- | -------------------- | ---------------------------------------- |
| **`^`** 在最外面      | **定位符** | 字符串的**开头**     | `^Cat` 匹配 "Cat is home"                |
| **`[^...]`** 在括号内 | **否定符** | **排除**括号里的字符 | `[^Cat]` 匹配除了 'C','a','t' 以外的字符 |

```python
non_letter = re.compile(r'[^a-z]') # 匹配任何非小写字母的字符

words[::-1]  # 反转序列
```
