# 滑动串口

### 要点

左指针移动时：

> **先更新状态，后移动指针**



## 定长滑动窗口模板

```python
def fixed_sliding_window(nums, k):
    n = len(nums)
    if n < k: return 0
    
    # 1. 初始化：计算第一个窗口的和
    current_sum = sum(nums[:k])
    max_sum = current_sum  # 假设题目是要求最大窗口和
    
    # 2. 开始滑动：从第 k 个元素开始，到最后一个元素
    for i in range(k, n):
        # 进一个 (nums[i])，出一个 (nums[i-k])
        current_sum = current_sum + nums[i] - nums[i-k]
        
        # 更新答案
        max_sum = max(max_sum, current_sum)
        
    return max_sum
```





## 不定长滑动窗口模板

```python
def sliding_window_template(nums, x):
    left = 0
    current_sum = 0
    count = 0
    n = len(nums)
    
    for right in range(n):
        # 1. 进窗口：将 nums[right] 加入统计
        current_sum += nums[right]
        
        # 2. 缩窗口：只要满足特定条件（如 >= x），就一直处理
        # 这里的核心是维护状态，直到条件不满足
        while current_sum >= x: 
            # 在这里处理你的逻辑（比如更新答案、记录次数）
            count += (n - right)
            
            # 3. 出窗口：移除 nums[left]，左指针右移
            current_sum -= nums[left]
            left += 1
            
    return count
```
