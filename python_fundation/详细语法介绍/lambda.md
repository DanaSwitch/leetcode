# lambda语法

### Lambda 语法详解

`lambda` 是 Python 中的**匿名函数**（即没有名字的临时函数）。它通常用于只需要用一次、逻辑简单的场景。

#### 基本语法结构：

Python

```
lambda 参数1, 参数2, ... : 表达式
```

- **`lambda` 关键字**：声明这是一个匿名函数。
- **参数**：可以有多个，用逗号隔开（就像普通函数的括号内容）。
- **冒号 (`:`)**：分隔参数和表达式。
- **表达式**：这是函数体。**注意：** Lambda 不需要写 `return`，它会自动返回表达式计算的结果。

------

#### 举几个例子对比：

**例子 A：普通的加法函数**

- 普通写法：

  Python

  ```
  def add(num):
      return num + 10
  ```

- Lambda 写法：

  Python

  ```
  add = lambda num : num + 10
  ```

**例子 B：在排序中使用（最常见场景）** 假设有一组学生数据：`students = [("张三", 18), ("李四", 20), ("王五", 15)]`（名字, 年龄）。 如果你想按**年龄**排序：

Python

```
students.sort(key=lambda x: x[1]) 
# 这里的 x 代表列表里的每一个元组，x[1] 就是年龄
```

**例子 C：多条件排序（也就是你代码里的写法）** 当我们需要先按一个标准排，冲突了再按另一个排，Lambda 可以返回一个**元组 `(tuple)`**。Python 在比较元组时，会按顺序比较：

Python

```
lambda x: (x[0], -x[1])
```

1. 先比 `x[0]`。
2. 如果 `x[0]` 一样，再比 `x[1]` 的负数。

------

### 3. 为什么在这里用 Lambda？

如果没有 Lambda，你得专门写一个函数给 `sort` 用，非常啰嗦：

Python

```
def my_rule(x):
    return (x[0], -x[1])

books.sort(key=my_rule)
```

使用 `lambda` 让代码变得非常紧凑，一行就能表达出复杂的排序逻辑。

**总结一下：** Lambda 就是一个“**即用即走**”的工具人函数，在处理 `sort`、`map`、`filter` 等需要传入“处理逻辑”的函数时非常方便。
